Demo

For more details to how to implement these examples, you can go to the documentation.

Examples #1

Autostarting, displaying the second image at loading. No link and no keys. Just standard controls.

Result
Code

$('#my_thumbs').desoSlide({
	main : {
		container : '#wizard_mainImage',
		cssClass  : 'img-responsive'
	},
	caption : true
});

Examples #2

Autostarting, displaying the second image at loading. No link and no keys. Just standard controls.

Result
Code

$('#ex1_thumbs').desoSlide({
	main : {
		container : '#ex1_mainImage',
		cssClass  : 'img-responsive'
	},
	auto : {
		start : true
	},
	first    : 1,
	effect   : 'roll',
	controls : {
		keys : false
	}
});

Examples #3

Displaying the overlay by hovering the main image. No link provided and no controls.

Result
Code

$('#ex2_thumbs').desoSlide({
	main : {
		container : '#ex2_mainImage',
		cssClass  : 'img-responsive'
	},
	effect   : 'flip',
	overlay  : 'hover',
	caption  : true,
	controls : {
		enable : false,
		keys   : false
	}
});
