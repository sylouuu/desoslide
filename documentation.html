At first, include jQuery and the desoSlide files in your page: 

<link rel="stylesheet" href="css/jquery.desoslide.css" />

<script src="js/jquery.js"></script>
<script src="js/jquery.desoslide.js"></script>

The CSS file is only needed for the caption styles.

Then, you need to create your thumbnails list with that markup. Don't forget to specify your images links and the alt attribute. 

<ul id="my_thumbs">
	<li>
		<a href="assets/img/bird.jpg">
			<img src="assets/img/bird_thumb.jpg" alt="Bird" data-caption="A bird, click me to open http://chez-syl.fr/" data-href="http://chez-syl.fr/">
		</a>
	</li>
	<li>
		<a href="assets/img/flower.jpg">
			<img src="assets/img/flower_thumb.jpg" alt="Flower" data-caption="A flower" data-caption="A flower, click me to open README.md file" data-href="README.md">
		</a>
	</li>
	<li>
		<a href="assets/img/butterfly.jpg">
			<img src="assets/img/butterfly_thumb.jpg" alt="Butterfly" data-caption="A butterfly" data-caption="A butterfly, click me to open LICENSE.md file" data-href="LICENSE.md">
		</a>
	</li>
	<li>
		<a href="assets/img/rose.jpg">
			<img src="assets/img/rose_thumb.jpg" alt="Rose" data-caption="A rose, you can't click me, got no link">
		</a>
	</li>
</ul>

Explanations:

  • Specify your images links and the alt attribute
  • The data-caption attribute is used for the caption (optional)
  • The data-href attribute contains the link to the page that will be active on the main image (optional)

Call the plugin "desoSlide" on your list of thumbnails by specifying the target where the main image will be generated: 

$(function() {
	$('#my_thumbs').desoSlide({
		mainImage: '#mainImage'
	});
});

Here, the main image will be generated inside #mainImage.

to see how to get that.

Examples

Displaying the second image at loading. Disabling the caption, no link and no keys.

  • Code
$('#ex1_thumbs').desoSlide({
	mainImage: '#ex1_mainImage',
	autoStart: true,
	firstImg: 1,
	enableCaption: false,
	enableControlsKeys: false
});

Displaying the caption by hovering the main image. No data-href provided and no control.

  • Code
$('#ex2_thumbs').desoSlide({
	mainImage: '#ex2_mainImage',
	displayCaption: 'hover',
	enableControls: false
});

Controls

You can control the slider by clicking on the pictos. Previous, pause, play and next are supported. Also, you are able to use your keyboard keys to control it:

  • left arrow: previous
  • right arrow: next
  • space: toggling play/pause

As this feature is binded on the document, if you have more than one slider on your page and you press a key, all the slider will be triggered. I recommand to you to enable the keys control on only one slider, by disabling the others with enableControlsKeys: false.

Documentation

Find below all the available options.

Options Values Default Description
mainImage Your target container false The main image container. If you don't specify a valid selector, you'll receive an error.
mainImageClass String 'img-responsive' for Boostrap 3 compliance Class(es) name(s) of you main image.
insertion 'append', 'prepend' or 'replace' 'append How to create the main image, by appending, prepending or replace all the selector's content (overwriting).
autoLoad Boolean true All thumbnails links (full size images) are auto loaded in background (invisible).
autoStart Boolean false Autostarting a diaporama.
firstImg An integer between 0 and the number of thumbnails (-1) 0 The index of the thumbnail that you want to display first.
enableCaption Boolean true Enable or not the main image caption. If true, be sure that you have specify the data-caption attribute on thumbnails.
displayCaption 'always' or 'hover' 'always' How to show the main image caption.
enableControls Boolean true Able to control the slider (prev/pause/play/next).
enableControlsKeys Boolean true Able to control the slider by using the keyboard shortcuts (left/right/space).
interval An integer, minimum 1500 3000 Interval in milliseconds between each image.
displayWarnings Boolean true Show warnings in the developer console. Be sure to put it false in production.
displayErrors Boolean true Show errors in the developer console. Be sure to put it false in production. If the render isn't ok, enable the errors then debug.
callback function(result) false Get the slider result. The first argument outputs one of these values: "success", "error" or "warning".

Retun value

The returned value by the .desoSlide() call is the jQuery object in order to maintain the chainability.

As explained before, if you want to know if the slider is well generated, use the callback function.

$('#ex2_thumbs').desoSlide({
	mainImage: '#ex2_mainImage',
	callback: function(result) {
		switch(result) {
			case 'success': // success handler
			break;
			case 'error': // error handler
			break;
			case 'warning': // warning handler
			break;
		}
	}
});
